# Ferns-and-Petals-Excel-Project

## Overview
This repository contains an interactive Excel dashboard developed to analyze the sales performance of Ferns N Petals (FNP). The dashboard summarizes revenue trends, customer spending patterns, product performance, and delivery timelines using PivotTables, Power Query, and dynamic slicers.

## Work Completed
- Cleaned and prepared the dataset using Power Query.
- Created derived fields such as month, hour, and delivery time difference.
- Developed PivotTables and PivotCharts for all analytical segments.
- Designed a structured dashboard with consistent formatting and slicers.
- Computed key metrics: total revenue, total orders, average spend, and average delivery time.
- Analyzed revenue by category, occasion, month, city, product, and hour.
